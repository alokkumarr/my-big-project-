7.3.1.4 Routes Actions

 

Figure 208 Available Routes

 

The Available Routes area of the Channel Management screen is located on the bottom half of the screen, it lists all available routes with the following information:

  • Route Name
  • Created By
  • Last Modified
  • File Pattern
  • Schedule
  • Source Location
  • Destination Location
  • Description
  • Actions
    • Activate
    • View Route Logs
    • Test Route Connectivity
    • Edit Route –The Details screen is populated to make changes (refer to Edit Route)
    • Delete Route

 

How to Activate Route

If the necessary route has not been activated, which can be viewed on the ‘Available Routes’ screen per the following steps:

 

Figure 209 Activate Route

  1. Upon accessing the 'Channel Management' Screen, Scroll down to the Available Routes section of the screen
  2. Locate the row to the Route from the list to be Activated and click on the Action icon ( ) at the end of that row to display the Action options
  3. Click on the ActivateIcon ()

How to Edit a Route

 

Figure 215 Edit Route

 

  1. Upon accessing the 'Channel Management' Screen
  2. Scroll down to the Available Routes section of the screen
  3. Locate the row to the Route from the list to be edited and click on the Action icon ( ) at the end of that row to display the Action options
  4. Click on Edit Route to display the Details Screen (Figure 186)
  5. Enter the details that need to be edited
  6. Click Next
  7. Make any scheduling changes that are necessary
  8. Click Update

 

How to Delete Route

 

Figure 216 Delete Route

  1. Upon accessing the 'Channel Management' Screen, scroll down to the Available Routes section of the screen
  2. Locate the row to the Route from the list to be deleted and click on the Action icon ( ) at the end of that row to display the Action options
  3. Click on Delete Route

 

Figure 217 Delete Row Verification

 

  1. A verification pop-up screen will be displayed with the "Route Name" to be deleted. Verify that it is the correct Route and Click on Yes ( ) to delete or No ( ) to leave the page without deleting.
