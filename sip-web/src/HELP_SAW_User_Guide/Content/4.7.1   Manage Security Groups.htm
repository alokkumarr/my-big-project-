4.7.1 Manage Security Groups

 

The Manage Security Groups area of Data Security option provides the following:

  • Lists of existing Security Groups
  • Capability to
    • Edit an existing Security Group
    • Delete an existing Security Group
    • Add a Security Group
    • Add Attributes to and existing or new

 

Figure 62 Manage Security Groups

 

How to Access Manage Security Groups

  1. Continue from Access Data Security
  2. From the expanded Data Security drop-down menu select Manage Security Groups

 

How to Edit and Existing Security Group

  1. Continue from Access Manage Security Groups
  2. Click on the edit icon ( ) to populate the Edit Security Group pop-up window

 

Figure 63 Edit Security Group

 

 

  1. Enter data into the fields:
    1. Security Group Name* (spaces are not allowed)
    2. Description

 

NOTE: Required fields are followed by an *

 

  1. Click on Save

 

How to Delete a Security Group

 

  1. Continue fromHow to Access Manage Security Groups
  2. Select the delete icon ( ) adjacent to the Security Group to be deleted. A pop-up screen to verify that the user is sure they want to delete the Security Group will appear

 

Figure 64 Delete Security Group Verification

 

  1. Select Delete or Cancel to terminate the deletion

 

How to Add a Security Group

 

Figure 65 Add Security Group

 

 

  1. Continue from How to Access Manage Security Groups
  2. Click on Add Security Group ( )
  3. Enter data into the fields:
    1. Security Group Name* (spaces are not allowed)
    1. Description
  1. Click on Create Group ( )
NOTE: Required fields are followed by an *

 

How to Add an Attribute

 

Figure 66 Add Attribute

 

  1. Continue from How to Access Manage Security Groups
  2. Click on Add Attributes ( )
  3. Enter the following:
    1. Field Name*
    2. Value*
  4. Click Create Attribute ( )

 

NOTE: Required fields are followed by an *

 

How to Delete an Attribute

 

  1. Continue from How to Access Manage Security Groups
  2. Select the delete icon ( ) adjacent to the Attribute to be deleted. A pop-up screen to verify that the user is sure they want to delete the Attribute will appear
