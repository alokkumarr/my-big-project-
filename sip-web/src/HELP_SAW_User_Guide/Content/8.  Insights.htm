8. Insights

Insights is a module within SAW that is available to only those users that has been granted access to the module. Ideally, the access is given by the admin user to reporting users only. It is a module that provides the option of selecting to view Advanced Analytics data via the Correlator or the Detector. The Correlator provides the user with a view of the available metrics and the mutual relation/connection between those metrics, while the Detector provides the user with a view of any anomalies that have occurred within the data. Anomalies can be values which are higher or lower than the normal expected value during the required time period.

How to Access the Insights Module

 

  1. Log into SAW

  1. Click on the Module Icon to display the modules available for selection
  2. Click on Insights to access the ‘Insights’ module. The default screen that is displayed is located within the 'Correlator' portion of the Insights module
