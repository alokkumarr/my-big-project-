8.1.1. Heat Map

 

 

The Heat Map view within the Correlator Viewer of the Insights module provides a graphical representation of the Correlation data.

Figure 221 Heat Map View

 

While in the Heat Map view the following options are available to the user to view and/or to select:

  • Filters that are selected
  • Filter icon ( ) – populates a screen with the following filter options available to select parameters to be populated:
    • Metric Selection Group
      • Dataset
      • Correlation Range
    • Metric Filter Group
      • X axis metrics
      • Y axis metrics
  • Download icon ( ) – provides the ability to down load the data represented in the Grid view to Excel.
  • Chart View icon ( ) – indicator that the user is view the Heat Map in a Chart view
  • Grid View icon ( ) – indicator that the user is viewing the Heat Map in a Grid view
  • Change Chart Pallet icon ( ) – provides color options for the Heat Map representation
  • Colored Squares – provides the correlation value and intersection of data.
    • Hover the curser over a square to view the details related to the intersection of the metrics of the X and Y axis from the correlation data
    • Click on the square to populate a Scatter view of the intersection of the selection
  • Chart Context Menu ( ) – provides the options to down load the following of the data”
    • Download PNG image
    • Download JPEG Image
    • Download PDF document
    • Download SVG vector image

How to Access the Heat Map

  1. Upon accessing the Insights Module (continue from How to Access the Correlator Area of Insights), click on Heat Map from the Correlation View area of the screen

 

How to Select the Data Parameters for the Heat Map

  1. Continue from “How to Access the Heat Map” How to Add a Description to a Report

Figure 222 Heat Map Select Filter Icon

 

  1. Click on the Toggle Filter Panel icon ( ) or on the current Filters.

 

Figure 223 Heat Map Filter Parameters

  1. Select the Dataset drop-down menu to display the options for selection:
    1. Chats
    2. Stickers
    3. Messages
    4. FileTransfers
    5. Subscribers
  2. Slide to enter the Correlation Range from-1 to 1
  3. In the Metric Filter Group
    1. Check the box(es) for the required metrics to be displayed on the X axis
    2. Check the box(es) for the required metrics to be displayed on the Y axis
  4. Click on Apply
