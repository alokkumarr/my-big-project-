8.2. Detector

 

 

The Detector area of the Insights Module provides the user with the Anomaly Viewer, where any anomalies within the selected metric are displayed for the selected data filter parameters. When the curser is hovered over the line of the line chart, scatter points are displayed with the following information:

  • Actual
  • Range
  • Date

The anomaly is highlighted in a red circle with an “A” for Anomaly ( ) and when hovered over displays:

  • Date
  • Expected Value
  • Actual Value

In addition, a decomposition chart is displayed with the following breakdowns:

  • Value vs Trend
  • Seasonal
  • Residual

The Anomaly Viewer screen provides the following:

  • Filters that are selected
  • Filter icon ( ) – populates a screen with the ‘Filter Data’ screen with options for selection as displayed in 8.2. Detector

 

NOTE:  Data choices are dependent on the Metric Filter Group option selected.

 

  • Download icon ( ) – provides the ability to down load the data represented in the Grid view to Excel.
  • Chart View icon ( ) – indicator that the user is viewing the Heat Map in a Chart view
  • Grid View icon ( ) – indicator that the user is viewing the Heat Map in a Grid view
  • Chart Context Menu ( ) – displays the options to down load the data in the following formats:
    • Download PNG image
    • Download JPEG Image
    • Download PDF document
    • Download SVG vector image

 

How to Access the Detector Area of Insights

 

Upon accessing the Insights Module (continue from “ section 8. Insights”),

  1. Click on the Menu icon ( ) from the upper left hand side of the screen to display the available areas of the Insights module
  1. Correlator (refer to section 8.1)
  1. Detector (refer to section 8.2)
  1. Click on Detector to display Anomaly Viewer
