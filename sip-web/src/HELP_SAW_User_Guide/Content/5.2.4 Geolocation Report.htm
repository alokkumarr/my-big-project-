5.2.4 Geolocation Report

 

The Geolocation functionality provides a geographical view of the data. Currently only the United States region by States is supported.

Figure 155 Geolocation View

 

How to Create a Geolocation Report

  1. From the Analyze module, select the +Analysis ( ) button

 

Figure 156 Geolocation Analysis Type

 

  1. Click on the ‘Analysis Type’ of Geolocation

 

Figure 157 Geolocation Type

 

  1. Click on the ‘Geolocation Type’ of Geolocation
  2. Click Next

 

Figure 158 Orders

  1. Select the type of 'Orders' data to include in the report
  2. Click Create

Figure 159 Geolocation Data Selection

 

  1. Select and drag the data to each area:
    1. Metrics
    2. Dimension
  2. Click on the drop-down to select additional filters for information:
    1. Amount of states to display
    2. Top or Bottom radio buttons for the region location
  3. Click on Apply Limit
  4. Click on the drop-down to select the global region
    1. Global region (currently only the United States)
  5. From the upper right hand side of the screen the user can click on:
    1. Preview to view the Geolocation map
    2. Sort to sort the data
    3. Filter to filter the data
  6. Click on Save from the upper left hand side of the screen to save the map.

 

How to Preview the Geolocation Map

 

  1. Continue from step 11 of How to Create a Geolocation Report

 

Figure 160 Preview Geolocation Map

  1. Click on Preview from the upper right hand side of the screen

Figure 161 Preview of Geolocation Map

  1. While in the Preview screen, the image can be enlarged by clicking on the + or – from the upper left hand side of the screen

 

How to Sort the Data on the Geolocation Map

 

 

  1. Continue from step 11 of How to Create a Geolocation Report

Figure 162 Geolocation Sort Option

 

  1. Select Sort from the upper right hand side of the screen

 

 

Figure 163 Sort Geolocation Data

 

 

  1. Move fields from ‘Available fields’ to ‘Sorted Fields’ or
  2. Move back to ‘Available fields’ from ‘Sorted Fields’ by clicking on the “X”
  3. Click on Apply Sort or Cancel

 

How to Filter a Geolocation Map

 

  1. Continue from step 11 of How to Create a Geolocation Report

Figure 164 Geolocation Filter Selection

  1. Select Filter from the upper right hand side of the screen

Figure 165 Filtering a Geolocation Report

  1. Select the order_data “+” to populate “Column”
  2. Click on ‘Column’ to display the available data fields
  3. Select the data fields to use for filtering
    1.  “Order” is displayed to add addition the following filter criteria:
      1. Equals
      2. Not equal
      3. Contains
      4. Starts with
      5. Ends with
      6. Is in

6. Select one of the following:

  1. Global
  2. Prompt (when selected “Optional” is displayed for selection)
  3. Optional
  4. “X” to remove the field

7. Select the “All” or “Any” “of the filters are true”

8. Click on Apply Filters to display the filtered map or Cancel
