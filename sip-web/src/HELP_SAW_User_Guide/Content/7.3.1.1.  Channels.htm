7.3.1.1. Channels

 

How to Create a Channel

 

Upon accessing the Channel Management Screen:

<login to Saw>Select Workbench module>Click on the Menu icon>Expand ‘Data Ingestion Service’>Select ‘Channel Management’ to populate the screen

  1. Click on the +Channel button

Figure 193 Data Sources to Create Data Channel

 

  1. Select the available data source (currently SFTP is the data source that is accessible through workbench)
  2. Click Next

Figure 194 Create Data Channel Details

 

  1. The Channel Details screen displays the following fields for data entry:
  1. Name of your Channel* - required field with a maximum of 20 characters allowed
  2. Hostname* - required field, IP address
  1. Port Number* - required field, SFTP currently on port 22
  2. Access Type – select Read
  3. Username* - required field, username to access the server
  4. Password* - required field, password to access the server
  5. Description of the source

 

NOTE: All required fields are marked with an *
  1. To verify the channel has been created correctly select Test Connectivity. Figure 184 displays a “Successful” test and Figure 185 displays a “Failed” test.

Figure 195 Test Connectivity Successful

 

Figure 196 Test Connectivity Failed

 

  1. Click on Create once the Connectivity Test is “Successful” to proceed with the creation of the Channel or Previous to return to the data sources step.
