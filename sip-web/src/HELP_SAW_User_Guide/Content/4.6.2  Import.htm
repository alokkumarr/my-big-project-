4.6.2 Import

 

The Import selection within the Package Utility option of the Admin menu provides a vehicle to Import an Analysis file. The screen provides the capability to Browse for the file and to select the Category where the file is to reside. In the event that any Errors occur an Error log will be populated.

 

Figure 57 Import Analysis Blank Screen

 

How to Import Analysis Files

  1. Continuing from the final step of How to Access Package Utility
  2. Select Package Utility to Expand the menu
  3. Select Import from the menu to populate the Import Screen
  4. Click on Browse ( ) and select the file to be imported

 

Figure 58 File Location

 

  1. Locate and select the file
  2. Click Open

 

Figure 59 Import File Selected

 

  1. The File will be populated on the right hand side of the screen with the details on the left hand side of the screen

 

NOTE: Files not folders must be selected. If the format is not accepted nothing will appear on the left hand side of the screen.

 

  1. Click on the Category drop-down arrow from the top of the screen to populate the available categories . The Category may also be changed from the details side of the screen (Category column) where the following information is populated:
    1. Analysis Name
    2. Analysis Type (e.g., report, chart, etc…)
    3. Metric Name
    4. Category
    5. Logs
  2. Select the Category where the file should reside
  3. The Category can be changed
  4. Check the box () adjacent to the file to be imported
  5. Click Import ( )

 

Figure 60 Successfully Imported

 

  1. The Log column indicates if the file was imported Successfully or not
  2. If a file with the same name already exists (Overwrite) will appear adjacent to the box
  3. Click on OVERWRITE ( ) to check the box
  4. Click on Import
  5. If Unsuccessful, click on Export Error Logs
