4.6.1 Export

 

The Export selection within the Package Utility option of the Admin menu provides a vehicle to export data and download the file. The screen provides a drop-down menu with additional drop-down menus that may categorize data files.

Figure 53 Export Screen

 

How to Export Data

  1. Continuing from the final step of How to Access Package Utility

Figure 54 Admin Menu Package Utility

  1. Select Package Utility to expand the menu

Figure 55 Export Screen with Category Selected

  1. Select Export from the menu to populate the Export Screen

Figure 56 Export Screen Populated

 

  1. Expand the 'Select Category' column to expand the categories where the data can be accessed
  2. Select the Category where the data will be found
  3. Select the files and check the box(es) of the file(s) to be exported. The files are moved to the right side of the screen

NOTE: More than one file can be checked for export
  1. Click on Export ( ), the file(s) to be exported are displayed at the bottom of the screen and are available in the users ‘Download’ folder
