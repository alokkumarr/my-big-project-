7.1.2. Add Dataset

 

Users with appropriate permissions will have the capability to Add a Data Set.

 

How to Add a Data Set

 

NOTE:  The steps are the same for adding both a Data Set or a Data Pod

 

  1. Log into Saw
  2. Click on the Module Icon to display the modules available for selection
  3. Click on Workbench, to access the ‘Workbench’ module
  4. Click on the menu icon from the upper left-hand side of the screen
  5. Click on ‘Datasets’ drop-down menu from the menu options
  6. Click on ‘Add Datasets’, the default screen will be displayed or from the View Datasets selection click on the button from the right-hand side of the screen to display the Staging Area screen

  2. Select the 'File mask'
