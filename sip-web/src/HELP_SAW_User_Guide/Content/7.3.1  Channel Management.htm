7.3.1 Channel Management

 

Channel Management functionality can only be performed by a System Administrator with appropriate permissions. The Channel Management screen provides the following functionality and information:

  • Channels of the selected data source that are available
    • These are external servers
  • Route information
  • +Channel
  • +Route
  • Edit Routes
  • Data Sources that are available for selection
    • Available data sources are highlighted with a blue outline also indicating the number of channels in the blue circle
  • Test Channel connectivity
  • Edit Channels
  • Delete Channels

How to Access Channel Management

 

  1. Log into Saw

Figure 190 Select Workbench Module

  1. Click on the Module Icon to display the modules available for selection
  2. Click on Workbench, to access the ‘Workbench’ module
  1. Click on the menu icon from the upper left-hand side of the screen
  2. Click on ‘Data Ingestion’ drop-down menu from the menu options
  3. Click on Channel Management

Figure 191 Select Workbench Module

  1. The menu is populated with the available options for selection:
  1. Datasets

• View Datasets

• Add Dataset

  1. Datapods

• Create Datapod

• Update Datapod

  1. Data Ingestion Service

• Channel Management

  1. Select Channel Management from the Data Ingestion Service area of the menu to display the ‘Channel Management’ screen
