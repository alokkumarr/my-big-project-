7.3.1.2. Routes

A Route must be created for each data set (i.e., Chat), the data is maintained for 7 days and when a new batch comes in the previous one becomes a .gz file. A search can be done at the higher level file to check all existing folders. Point to the mapped cluster followed by /raw.

 

How to Create a Route

Upon accessing the Channel Management Screen:

<login to Saw>Select Workbench module>Click on the Menu icon>Expand ‘Data Ingestion Service’>Select ‘Channel Management’ to populate the screen

 

  1. Click on the +Route button

Figure 197 Create Route Details Screen

 

Figure 198 Entry Error – Route Name Already Exists

 

 

NOTE: If the Route Name already exists the box will be outlined in red and an error message will be displayed “Route Name already exists” in red.
  1. Fill in the information for all required fields:
    1. Route Name*
    2. File Pattern*
    3. Exclude Extension
    4. Source Location*
    5. Destination Location*
    6. Batch size*
    7. Description about the route
  1. Click on Test Connectivity ( ) to ensure connectivity is ‘Successful’

If the connection fails, then review the data that has been entered on the details screen including source and destination
