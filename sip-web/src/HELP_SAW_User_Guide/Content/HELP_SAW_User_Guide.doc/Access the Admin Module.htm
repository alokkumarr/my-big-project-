4.1. Access the Admin Module

 

How to Access the Admin Module

  1. Click on the Module Icon to display the modules available for selection

  2. Click on Admin to populate access the Admin Module and display the default ‘User Management’ screen

 

NOTE: Only System Administrators with Admin Role privileges will have the Admin module option available to them.
