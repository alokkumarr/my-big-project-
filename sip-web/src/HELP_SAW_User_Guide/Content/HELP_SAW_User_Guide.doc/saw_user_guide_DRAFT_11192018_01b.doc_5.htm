4.5. Category Management

 

Categories and Sub-Categories are used to organize and store the analysis.  A Category is a method of grouping Sub-Categories.  Administrators configure Privileges related to different Roles to determine access to Categories and Sub-Categories.  The Category Management screen (Figure 31  Category Management Screen) contains the following:

Columns:

  • Product – the product associated with the Category
  • Module – determines the module that the Category should appear
  • Category – a Category associated with the Category (e.g., Daily Data)
  • Sub Categories – associated with the Category (e.g., Drafts)
