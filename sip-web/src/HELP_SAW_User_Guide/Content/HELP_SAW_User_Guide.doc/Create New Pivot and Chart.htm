5.2.2. Create a New Chart

 

Creating a new chart requires a user with the appropriate permissions to ability to access the Analyze module (refer to 5. Analyze Module for instructions).

 

How to Create a New Chart

To create a new chart:

  1. From the Analyze module, select the +Analysis ( ) button

 

Figure 117 Create New Analysis_Chart

 

Figure 118 Create New Analysis

 

  1. Click on Chart form the ‘Select Analysis type’ area of the screen

  2. Select the type of graphical representation to display from the following located in the ‘Select Chart type’ area of the screen

    • Column Chart – Displays vertical bars going across the chart horizontally and the values axis displays on the left side of the chart. Refer to "Column Chart" in section 5.2.2.1. Column Chart
    • Bar Chart – A type of bar graph that shows horizontal or vertical bars going across the chart horizontally and the values while on the bottom of the chart. Refer to "Bar Chart" in section 5.2.2.2. Bar Chart
    • Stacked Chart – Plot data sets to display on top of each other with the cumulative magnitude of two or more data series. Refer to "Stacked Chart" in section 5.2.2.3. Stacked Chart
    • Line Chart – A type of line graph that is plotted using a series of lines going across the chart horizontally and the values axis displays on the left side of the chart. Refer to "Line Chart" in section 5.2.2.4. Line Chart
    • Area Chart – Displays graphically quantitative data and is based on the line chart. The area between the axis and line are emphasized with colors, textures, and hatchings. Use to compare with an area chart of two or more quantities. Refer to "Area Chart" in section 5.2.2.5. Area Chart
    • Combo Chart – Displays a visualization that combines the features of the bar chart and the line chart. The combination chart displays the data using bars and/or lines, each of which represent a category. Refer to "Combo Chart" in section 5.2.2.6. Combo Chart
    • Scatter Plot – Use horizontal and vertical axes to plot data points to show how much one variable is affected by another and their correlation or relationship between two variables. Refer to "Scatter Plot" in section 5.2.2.7. Scatter Plot
    • Bubble Chart – Displays three dimensions of data. Each entity with its triplet (v1, v2, v3) of associated data is plotted as a disk that expresses two of the vi values through the disk's xy location and the third through its size. Refer to "Bubble Chart" in section 5.2.2.8. Bubble Chart
    • Pie Chart – Displays quantitative information by means of a circle divided into sectors, in which the relative sizes of the areas (or central angles) of the sectors correspond to the relative sizes or proportions of the quantities. Refer to "Pie Chart" in section 5.2.2.9. Pie Chart
    • Time Series Chart – refer to section 5.2.2.10. Time Series Chart
    • Time Series Multi-pane – refer to section 5.2.2.11. Time Series Multi-Pane
  1. Select Create Analysis

NOTE:  The data elements display depending on what type of Chart that has been selected AND the type of data available from the data lake

 

  1. Select the Data Fields: Metrics, Dimensions, and Grouped By. This is the data acquired from the data lake from which you can populate a report.

Chart Options differ depending on the previous selections:

  1. Select Inverted or Not Inverted to determine the origin of how certain reports display. Refer to section 5.1.5. Inverted or Not Inverted
  2. Select the Legend Options which is the location where the legend displays: Left, Right, Top, or Bottom.
  3. Select the Layout which determines the direction the report displays on the page: Vertical (portrait) or Horizontal (landscape).

Use the options in the upper right corner of the screen to further define your report

  1. Select Refresh Data to populate or update the report. Refer to section 5.1.7. Refresh Data
  2. Enter a Description for the report. Refer to section 5.1.6
  3. Select Preview to view the report. Refer to section 5.1.8. Preview Report
  4. Select Sort define order of the data. Refer to section 5.1.13. Save Analysis
  5. Select Filter to categorize the data for the report. Refer to section 5.1.9. Filter
  6. Select the origin of how certain reports display. Refer to "Inverted or Not Inverted" in section 5.1.5. Inverted or Not Inverted
  7. Select Save. Refer to section 5.1.3. Search.
