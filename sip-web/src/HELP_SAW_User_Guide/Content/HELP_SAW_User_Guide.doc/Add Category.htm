4.5.1. Add Category

 

A Category provides a way to group Sub-Categories.  The screen provides the option to add a Category or Sub-Category.  The Category options provides the following fields (all fields marked with an * are required):

  • Product* - product to be associated to the Category
  • Module* – defines which module the Category should appear
NOTE: Once the product is selected Module field is required (*)
  • Category Name* - Category Name to reflect the name of what this Category is going to collect or group in the Sub-Category.
  • Description – detailed description of the Category
  • Status* - ACTIVE or INACTIVE

 

How to Add a Category

To add a Category, perform the following:

  1. Upon accessing the Category Management screen (refer to 4.5, Category Management)
  2. Click + Category ( )
  3. Click on the Category button
  4. Enter information into all required fields
