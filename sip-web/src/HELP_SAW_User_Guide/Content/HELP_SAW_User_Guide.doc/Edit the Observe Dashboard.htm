6.1. Edit the Observe Dashboard

 

The Observe dashboard can be updated and changed by selecting Edit.  From this screen the user can move widgets (graphics) around, Add widgets, Filter widgets, Delete widgets, and Save the changes that were made.

 

How to Edit the Observe Dashboard

  1. Log into Saw
  2. Click on Observe, from the upper middle of the screen
  3. Click on Edit from the top menu of the Dashboard to populate the dashboard for editing

 

Figure 102 Observe Top Menu

 

How to Move Widgets on the Observe Dashboard

1. Log into Saw

2. Click on Observe, from the upper middle of the screen

3. Click on Edit from the top menu of the Dashboard to populate the dashboard for editing

4. Click on the widget and drag to the position that you would like it to be.

 

How to Increase the Size of a Widget

Figure 103 Expand Widget Size

 

  1. Log into Saw
  2. Click on Observe, from the upper middle of the screen
  3. Click on Edit from the top menu of the Dashboard to populate the dashboard for editing
  4. Click on the widget and drag to the lower-right hand corner to the size position that you would like it to be

 

How to Delete a Widget

Figure 103 Delete Widget

 

  1. Log into Saw
  2. Click on Observe, from the upper middle of the screen
  3. Click on Edit from the top menu of the Dashboard to populate the dashboard for editing
  4. Click on the “X” in the upper right-hand corner of the widget to delete

 

How to Add a Widget
