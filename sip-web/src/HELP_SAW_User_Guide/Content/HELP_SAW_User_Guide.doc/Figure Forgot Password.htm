2.4. Change SAW Password

 

How to Change Your SAW Password

  1. Select the Account Settings icon in the upper right corner of the screen (this represents the system admin role)
  2. Select Change Password as displayed in Figure 6 Change Password Details

 

Figure 5  Change Password

  1. Enter your Old Password from the ‘Change Password’ screen displayed in Figure 6 Change Password Details

    Figure 6 Change Password Details

NOTE: If you forgot your Old Password refer to 2.3. Reset Password

 

  1. Enter the New Password on the ‘New Password’ line
  2. Confirm the new Password by re-entering it on the ‘Confirm Password’ line
  3. Click on Change to use your new password
  4. Or click on Cancel to delete changes and log in with your old password
