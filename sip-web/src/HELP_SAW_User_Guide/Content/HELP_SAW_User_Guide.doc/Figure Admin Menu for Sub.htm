4.2. User Management

 

In this instance ‘Users’ are the individuals that are assigned or are to be assigned access and permissions to SAW by the SAW System Administrator.   They may have different Roles and permissions either limiting or enhancing their access to the application functionality and data.

 

How to Access the User Management Screen

 

  1. The System Admin selects the Account Settings icon in the upper corner of the screen
  2. From the drop-down menu select Admin. By default, the User Management screen displays (proceed to step 5), it does not display proceed to step 3

Figure 14 Admin Drop-Down Menu

  1. Select the menu icon from the ‘User Management’ screen to access the User Management sub-modules
  1. Select the User option

Figure 15 User Selected

  1. From this menu the System Admin has the capability to perform the following:
  • Add a user – select
  • Delete a user – select
  • Edit a user – select
  • Sort the columns on the screen
  • Access additional screens

Figure 16 User Management Screen

 

NOTE: Select the column header to change the sort order of any of the columns on the screen
