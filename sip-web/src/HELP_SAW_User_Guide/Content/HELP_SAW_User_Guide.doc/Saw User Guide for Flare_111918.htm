1. SAW Visualization Overview

The Synchronoss Analytics Workbench (SAW) application is part of the Synchronoss Insights Platform (SIP) that provides analysis and visualization from information collected from the data lake.

SIP is an analytics platform that offers end to end features that are required to generate advanced insights.  Built from the ground up using big data technologies, SIP offers the fastest mechanism to ingest, process, model and visualize analytic insights. 

SAW is the portal that integrates with SIP to provide users a quick way to prepare, analyze or model, and visually report on key insights and Key Performance Indicators (KPIs) from the collected data .  SAW enables the user to analyze and view the massive amounts of data from the data lake quickly and with low latency rates.

The SAW platform uses real-time and batch data ingestion, provides workflow and data preparation through a data pipeline, while applying descriptive and predictive analytics to provide visualization through interactive dashboards and adhoc analysis.

When SAW is fully matured it is planned to have an Alert module with the capability to notify users when certain Key Performance Indicators (KPIs) reach a specific threshold (above or below a designated level). This Alert module would be a key benefit to the Operations Team and Business Owners to provide the anomalies or outliers resulting from the data.

The Synchronoss Analytics Workbench (SAW) application includes modules that will be accessible depending on the user role.  Those users with authorized privileges may have access to one or more of the following modules:

Modules
Module Description
Admin

Administrator only module.

Administrators also configure Categories and Sub-Categories to organize and store the analysis.

Analyze

Used to perform data exploration and analysis by an authorized user.

Administrators configure Categories and Sub-Categories to organize and store the analyses.

Provided data sets can be visualized in one of the three methods: Tabular Report, Pivots and Charts.

Observe The Observe includes interactive dashboards populated from the data from the Analyze module. It is also referred to as Premium Insights. This module provides Executives, decision makers, and others use the capability to view the overall trend in growth and success of the product. Can be set up to be the landing page with selected information.
Workbench Contains the data sets and data pods that are used for analytics
