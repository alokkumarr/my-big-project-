4.3.2. View/Edit Role Details

 

How to View/Edit Role Details

  1. From the Role Management screen (refer to 4.3.1. Access Role Management
  2. Click on edit icon ()to review or edit a Role.  From this screen the following fields can be viewed/changed:

Figure 24 Edit Role Screen

  1. Review or make changes on the 'Edit Role' screen
  2. Select Save to keep the changes or Cancel to leave the page without making any changes
  1. Role name* (no spaces, under score can be used)
  2. Description*
  3. Status* (ACTIVE or INACTIVE)
  4. Role Type* (Admin or USER)
NOTE: Once the fields have been populated the Create Role button becomes activated
  1. Select "Create default privileges for My Analysis Category" to automatically grant full permissions for the new role to all sub-categories in the “My Analysis” category
  2. Click Create Role to save the data or Cancel to leave the page without making any updates
