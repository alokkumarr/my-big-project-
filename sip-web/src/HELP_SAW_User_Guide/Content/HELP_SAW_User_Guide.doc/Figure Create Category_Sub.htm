4.5.5. Edit Sub-Category

 

How to Edit a Sub-Category

To edit a Sub-Category, perform the following:

 

NOTE: There isn’t a separate selection on the Category Management screen for a Sub-Category. Selecting the edit icon will populate the Edit Category screen where a Category or Sub-Category can be updated.
  1. Access the 'Category Management' screen by referring to section 4.5. Category Management
  2. Select the Edit icon ( ) from the row of the Sub-Category to be updated
  3. Selecting the "Sub-Category" to be edited
  4. The following fields can be updated if necessary:
  1. Change the Product to associate with the Privilege
  2. Change the Module defining in which module the Category should appear
  3. Change the Category Name assigned to the Privilege. Refer to section 4.5 Category Management
  4. Change the detailed Description of the Category
  5. Change the Category's Status to Active or Inactive
  6. Change the Sub-Category Name assigned to the Sub-Category, such as Drafts. Refer to section 4.5 Category Management

 

NOTE: The Sub-Category only supports alpha characters.
  1. Change the detailed Description of the Sub-Category
  2. Change the Sub-Category's Status to Active or Inactive
  1. Select Save.
