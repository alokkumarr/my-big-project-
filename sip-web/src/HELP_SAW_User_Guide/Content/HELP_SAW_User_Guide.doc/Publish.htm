5.1.14.4. Publish Analysis

 

Use to publish the data to a shared sub-category. Publish enables you to move an item to a different sub-category.  Publish also gives you the ability to create, edit, remove a schedule for automatic recurring execution.

 

Figure 100 Publish

 

How to Publish an Analysis

  1. Select the Options ( ) next to the report you desire to change categories
  2. Select Publish from the Options on the row of the analysis
  3. Select the Category under which to Publish this analysis
  4. Select Publish.
