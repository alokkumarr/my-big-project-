5.1.1. List View

The List View option is used to display the reports that are available in a list on the screen.

 

How to Open List View

  1. Access the Create Analysis screen by referring to section 5.1
  2. Select the List View icon

 

Figure 51 Analyze Module List View

 

 

 

The List mode in the Analyze module consists of the following fields:

  • Name – Name of the analysis
  • Metrics – Where the analysis was collected
  • Scheduled – Date on which the analysis is collected
  • Type – Type of analysis display: All, Charts, Reports, or Pivots
  • Creator – Who created the analysis
  • Created – Date the analysis was created
  • Report Options – Refer to section 5.1.14
