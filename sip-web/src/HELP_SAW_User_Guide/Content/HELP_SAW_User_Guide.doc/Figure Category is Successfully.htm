4.5.6. Delete Sub-Category

 

How to Delete a Sub-Category

To delete a Category, perform the following:

  1. Access the Category Management screen by referring to section 4.5. Category Management
  2. Select the Delete icon () from the row of the Sub-Category to be deleted

 

Figure 49 Delete Sub-Category

 

  1. Click the Delete Icon () of the Sub-Category to be deleted or Cancel to terminate the action
  2. A successful delete message is displayed in the upper right-hand corner of the screen.
