7. Workbench Module

 

The Workbench contains several options available that the user can utilize:

 

  • View Data Sets in either in a list mode () or a card mode ()
  • View Data Pods in a list mode () or a card mode ()
  • The user can decide if they would like the data to be Auto Refreshed by selecting the toggle switch () or not
  • Datasets or Data Pods can be added by clicking +Dataset
  • Search () for data by entering the data set name in the field

How to Access the Workbench and Setup the Screen

  1. Log into Saw
  2. Click on Workbench, from the upper middle of the screen
  3. Perform any of the following:
    1. View Data Sets by clicking either the list icon or a card icon
    2. View Data Pods by clicking either the list icon or a card icon

Refresh Data Automatically

    1. Select the toggle switch and slide to the right to Automatically refresh Data

NOTE:  The steps are the same for adding both a Data Set or a Data Pod
