7. Workbench Module

The functionality within the Workbench Module is only accessible by users with appropriate permissions. The module will only display those areas that the user that logged into the SIP Portal is authorized to access.

The Workbench contains several options available that the user can utilize:

Figure 175 Workbench

 

 

How to Access the Workbench Module

  1. Log into Saw
  2. Click on the Module Icon to display the modules available for selection
  3. Click on Workbenchfrom the modules displayed

 

How to Access the Workbench Menu

While in the Workbench Module:

1. Click on the Menu Icon from the upper left-hand side of the screen

2. Click on the drop-down arrows to display menu items for selection

The workbench menu options contain different areas of access; however, only menu items that have been assigned to the logged in user will be displayed.

Figure 176 Workbench Sub-Menu

 

The sub-menu includes 3 main areas that can be expanded to display additional areas to access as shown in 7. Workbench Module

  • Datasets includes:
    • View Datasets
    • Add Datasets
  • Datapods includes:
    • Create Datapod
    • Update Datapod
  • Data Ingestion Service includes:
    • Channel Management (details related to Channel Management can be found in the RCS Analytics V1.5 Deployment and Operations Manual
