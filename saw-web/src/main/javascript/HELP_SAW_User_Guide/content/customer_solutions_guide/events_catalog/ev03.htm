Event Type EV03

Event Type EV03 is the Order Submission (Multiple Transaction) code that indicates multiple events were generated when a New Order is submitted is submitted through the Order Gateway.

Each subscriber or device is associated with EV01. Refer to Event Type Word Files to print event tables.
