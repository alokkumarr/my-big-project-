Privilege Management

Administrators configure Privileges to different Roles to determine their access to Categories and Sub-Categories.

To view the Privilege Management screen:

  1. Select the Account Settings icon in the upper corner of the screen.
  2. Select Admin.
  3. Select Privilege from the Manage menu.

Privilege Management

Note: Select the column header to change the sort order.

Privilege – Select to view current Privilege information.

  • Product - Product associated with the Privilege.
  • Module - Analyze module creates the analysis and analyze the data. Observe module populates data from the Analyze module.
  • Category - Category associated with the Privilege. Refer to Category Management.
  • Sub Categories - Sub Category associated with the Privilege. Refer to Category Management.
  • Role - Current Role assigned to the Privilege: Admin (Administrator) or User level. Refer to Role Management.
  • Privilege Desc – Detailed description of the Privilege.
  • +Privilege – Select to add a new Privilege. Refer to Add Privilege.
  • Edit Privilege – Select to edit a Privilege. Refer to Edit Privilege.
  • Delete Privilege – Select to delete a Privilege. Refer to Delete Privilege.
