Create New Pivot and Chart

To create a new chart:

  1. Select the +Analysis button.

New Analysis

  1. Select the data to analyze from the What would you like to analyze radio buttons. These options are customer specific and depend on what you are collecting from your data lake.
  1. Select the type of Pivot Table to button.
  1. Select the type of Chart to display graphical representation of your data.
  • Column Chart – Displays vertical bars going across the chart horizontally and the values axis displays on the left side of the chart. Refer to Column Chart.
  • Bar Chart – A type of bar graph that shows horizontal or vertical bars going across the chart horizontally and the values while on the bottom of the chart. Refer to Bar Chart.
  • Stacked Chart – Plot data sets to display on top of each other with the cumulative magnitude of two or more data series. Refer to Stacked Chart.
  • Line Chart – A type of line graph that is plotted using a series of lines going across the chart horizontally and the values axis displays on the left side of the chart. Refer to Line Chart.
  • Area Chart – Displays graphically quantitative data and is based on the line chart. The area between the axis and line are emphasized with colors, textures, and hatchings. Use to compare with an area chart of two or more quantities. Refer to Area Chart.
  • Combo Chart – Displays a visualization that combines the features of the bar chart and the line chart. The combination chart displays the data using bars and/or lines, each of which represent a category. Refer to Combo Chart.
  • Scatter Plot – Use horizontal and vertical axes to plot data points to show how much one variable is affected by another and their correlation or relationship between two variables. Refer to Scatter Plot.
  • Bubble Chart – Displays three dimensions of data. Each entity with its triplet (v1, v2, v3) of associated data is plotted as a disk that expresses two of the vi values through the disk's xy location and the third through its size. Refer to Bubble Chart.
  • Pie Chart – Displays quantitative information by means of a circle divided into sectors, in which the relative sizes of the areas (or central angles) of the sectors correspond to the relative sizes or proportions of the quantities. Refer to Pie Chart.
  1. Select Create Analysis.
Note: The data elements display depending on what type of Chart you previously selected AND the type of data available from your data lake.
  1. Select the Data Fields: Metrics, Dimensions, and Grouped By. This is the data acquired from the data lake from which you can populate a report.

Chart Options differ depending on the previous selections:

  1. Select Inverted or Not Inverted to determine the origin of how certain reports display. Refer to Inverted or Not Inverted.
  2. Select the Legend Options which is the location where the legend displays: Left, Right, Top, or Bottom.
  3. Select the Layout which determines the direction the report displays on the page: Vertical (portrait) or Horizontal (landscape).

Use the options in the upper right corner of the screen to further define your report.

  1. Select Refresh Data to populate or update the report. Refer to Refresh Data.
  2. Enter a Description for the report. Refer to Description.
  3. Select Preview to view the report. Refer to Preview.
  4. Select Sort define order of the data. Refer to Sort.
  5. Select Filter to categorize the data for the report. Refer to Filter.
  6. Select the origin of how certain reports display. Refer to Inverted or Not Inverted.
  1. Select Save. Refer to Save Analysis.
