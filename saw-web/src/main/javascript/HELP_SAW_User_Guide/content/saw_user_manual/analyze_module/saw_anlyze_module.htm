SAW Analyze Module

The Analyze module enables you to create the analysis and analyze the data. The Analyze module visualizes the data sets using one of the three methods: Tabular Report, Pivots and Charts. You use the Analyze module to perform data exploration and analysis. Use the Analyze module to view saved reports and create new reports.

The Administrator configures the Analyze model, they set up Users who access SAWSynchronoss Analytics Workbench. Administrators configure Categories and Sub-Categories to organize and store your analyses. A Category is a way to group Sub-Categories. Administrators set Privileges to different Roles to determine their access to Sub-Categories.

Analyze Module Example

Analyses Menu Displays the list of Category analysis available for reporting.
Use the Menu icon to expand and collapse report views and lists.
Use the Search field to locate the files you desire to view. Refer to Search.
Use the Card View icon to view reports as charts. Refer to Card View.
Use the List View icon to view reports as lists. Refer to List View.
Use the Types drop-down list to select the specific type of reports to view.
Use the +Analysis button to start a new report.
Note: Select the column header to change the sort order of the tables or hover over a column line to increase or decrease the size of the column.
