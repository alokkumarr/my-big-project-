Fork

Use Fork to clone or copy an existing analysis to edit the data and create a new analysis.

  1. Select the Fork icon next to the analysis you desire to copy.

  1. Select the Dataset Fields to display in your report.
  2. Select the Chart Options and select Apply.
  • Show Labels enables you to display chart values as a percentage (Show Percentage) or as a value (Show Data Value).

Note: Select the Refresh data button to update the analysis with the newest data.
  1. Select Description to provide a detailed and meaningful definition of what the report analysis displays.
  2. Select Preview to see your report in full screen view.
  3. Select Sort to determine the order in which data displays in the report.
  4. Select Filter to include or exclude data in the report.
  5. Select Save. Refer to Save Analysis.
