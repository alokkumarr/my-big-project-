Deploy XDF

eXtreme Data Framework also called XData Framework. The XDF framework provides a dashboard and drill-down reporting solution for SAWSynchronoss Analytics Workbench. Events are ingested real-time using Real Time Ingestion Service (RTIS) and Real Time Processing Service (RTPS) frameworks and then the XDF pipeline framework processes the data. After this installation, the XDF data loading pipeline activates and data enters the ElasticsearchElasticsearch is a distributed, RESTful search and analytics engine capable of solving a growing number of use cases. Elasticsearch centrally stores data for discovery. index.

Note: Install and deploy XDF on EACH node.

To install XDF as the mapr user:

  1. Run su - mapr and press Enter.
  2. Run mkdir /dfs and press Enter on EACH MapR node.
  3. Mount /dfs on EACH MapR node before installing the package. Run mount <masternode>:/mapr/<clustername>/ /dfs and press Enter.

    4. For example, if the <masternode> is called mapr-sand01 and the <clustername> is called sandbox, run mount mapr-sand0 /mapr/sandbox/ /dfs and press Enter.
