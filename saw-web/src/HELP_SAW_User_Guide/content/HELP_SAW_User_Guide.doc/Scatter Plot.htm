5.2.2.7. Scatter Plot

 

Scatter Plots use horizontal and vertical axes to plot data points to show how much one variable is affected by another and their correlation or relationship between two variables.

 

How to Create a Scatter Plot

  1. From the 'Analyze' module, Select the +Analysis ( ) button

 

Figure 105 Scatter Plot Selected

  1. Select Chart from the ‘Select Analysis Type’ options

  2. Select Scatter Plot to display graphical representation of your data from the ‘Select Chart type’ options

  3. Select the Next button

 

Figure 106 Select the Datapod for a Line Chart

 

  1. Select the 'Datapod' to analyze from available options (e.g., Default, orders). These options are customer specific and depend on what you are collecting from your data lake

NOTE: The data elements display depending on what type of Chart you previously selected AND the type of data available from your data lake.

 

  1. Click Create

 

Figure 107 Scatter Plot

 

  1. Select the Data Fields: Metrics, Dimensions, and Grouped By. This is the data acquired from the data lake from which you can populate a report

Chart Options differ depending on the previous selections. Refer to section 5.1. Analysis Options

  1. Select Save. Refer to section 5.1.13. Save Analysis.
