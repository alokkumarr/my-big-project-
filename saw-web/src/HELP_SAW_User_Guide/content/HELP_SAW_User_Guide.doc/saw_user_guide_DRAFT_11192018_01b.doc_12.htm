5.1.14.6. Schedules

 

Used to schedule reports on a specific schedule.  To Schedule Analysis designate the:

  • Scheduling Interval or to schedule immediately
  • Output Format
  • Publish to Email (Click on the field to make available for input, Emails are to be separated by ‘;’)
  • FTP Location (select from drop-down)

 

Figure 72 Schedule

 

NOTE:    This screen is also used to Remove Schedule

 

How to Schedule a Report to Run

  1. Select the Options ( ) next to the analysis to be scheduled
  2. Select Schedule from the Options
  3. Select if the report is supposed to run Immediately, Hourly, Daily, Weekly, Monthly, or Yearly

How to Remove a Schedule from a Report

Figure 80 Remove Schedule
