5.1.4. View Analysis by Type

 

The All Types option displays all the available analysis by types through a drop-down selection. Use the All Types drop-down list box to limit the results of the analysis displayed by either selecting All, Chart, Report, Pivot, or Scheduled (Figure 55) to be displayed on the screen.

 

How to View Analysis by Type

  1. Access the Create Analysis screen by referring to section 5.1
  2. Select the All Types drop-down list box

 

Figure 54 All Types

 

  1. Select the type of data to display: Charts, Reports, Pivots, or Scheduled data

 

Figure 55 All Types Drop-Down

 

NOTE:  Use Search and All Types together to produce better results.
