5.1.14.3. Edit a Report

 

Use to edit the report analysis data by performing the following:

 

How to Edit a Report

  1. Select the Options () next to the report to be edited
  2. Select Edit
  3. Delete Filters Applied if necessary
  4. Select the Data Fields: Metrics, Dimensions, and Grouped By (this is the data acquired from the data lake from which you can populate a report)

Chart Options differ depending on the previous selections

  1. Select Inverted or Not Inverted to determine the origin of how certain reports display. Refer to "Inverted or Not Inverted" in section 5.1.5. Inverted or Not Inverted .
  2. Select the Legend
  3. Options which is the location where the legend displays: Left, Right, Top, or Bottom.
  4. Select the Layout which determines the direction the report displays on the page: Vertical (portrait) or Horizontal (landscape).

Use the options in the upper right corner of the screen to further define your report

  1. Select Refresh Data to populate or update the report. Refer to section 5.1.7. Refresh Data.
  2. Enter a Description for the report. Refer to section 5.1.6. Description
  3. Select Preview to view the report. Refer to section 5.1.8. Preview Report
  4. Select Sort define order of the data. Refer to section 5.1.10. Sort
  5. Select Filter to categorize the data for the report. Refer to section 5.1.9. Filter
  6. Select Save. Refer to section 5.1.13. Save Analysis.
