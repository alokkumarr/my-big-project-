4.2.2. Edit/View User Information

 

How to Edit/View User Information

 

From the User Management screen (refer to How to Access User Management)

  1. Click on the ‘edit’ icon (Pencil ) on the row of the user to be edited or reviewed. The following fields are displayed on the Edit User screen:
  1. Role* - Assigned user role
  2. First name* - First name of user
  3. Middle name
  4. Last name* - Last name of user
  5. Login ID* - Login identification for the user
  6. Password - Enter a generic password and have users change their passwords upon login

NOTE: Passwords must have a minimum of 8 alphanumeric characters with 1 uppercase letter and 1 special character.
  1. Email* - User’s email address
  2. Status* - current status of the user must be selected (Active or Inactive)

NOTE: All fields marked with an * are required

2. Make the necessary updates ensuring that all required (*) fields contain data

Figure 19 Edit User Screen

3. Click to Save data or Cancel to abandon screen.
