5.1.12. Fork

 

Use Fork () to clone or copy an existing analysis to edit the data and create a new analysis. 

  1. Select the Fork icon () next to the analysis you desire to copy
  1. Select the Dataset Fields to display in your report
  1. Select the Chart Options and select Save
  1. Show Labels enables you to display chart values as a percentage (Show Percentage) or as a value (Show Data Value).
