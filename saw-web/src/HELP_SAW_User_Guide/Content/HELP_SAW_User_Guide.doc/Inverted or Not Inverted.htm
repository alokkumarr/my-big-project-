5.1.5. Inverted or Not Inverted

 

The Inverted and Not Inverted is available for Charts to determine the origin of a chart and if it will be represented vertically or horizontally.  This option is available for existing charts and for new charts being created.

 

How to Access the Inverted or Not Inverted Option on an Existing Chart

  1. Access the Create Analysis screen by referring to section 5.1
  2. Access Chart from the All Types drop-down list to produce a list of charts for selection
  3. Select an existing Chart to change the origin
  4. Select the Inverted or Not Inverted option from the top left hand-side of the chart screen
