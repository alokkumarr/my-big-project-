4. Administrator Module

 

System Administrators are the only SAW users with access to the Admin module.  This module also has access to the sub-modules listed in Table 4  Administrator Module Sub-Modules .  The sub-modules can be accessed by clicking on the menu icon in the top left-hand side of the screen next to the Synchronoss logo.

 

Table 4 Administrator Module Sub-Modules
Admin Sub-Modules Description
User Management Add, Delete, Update Users
Role Management Add, Delete, Update Roles
Privilege Management Add, Delete, Update Privileges
Category Management Add, Delete, Update Categories
Package Utility Import and Export Package Utilities
