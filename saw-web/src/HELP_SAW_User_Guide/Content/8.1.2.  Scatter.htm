8.1.2. Scatter

 

The Scatter view of the Insights module provides a graphical representation in the form of a Correlation time series chart of the data in addition to a Correlation Scatter chart. The scatter chart is a linear regression line drawn through the metrics values being correlated.

The chart displays:

Filters that have been selected and applied to the x and y axis

The user can click on these filters to populate a screen with filter options

  • Filter icon ( ) – populates a screen with the following filter options available to select parameters to be populated:

    • Metric Selection Group
    • Dataset
    • X axis metric
    • Y axis metric
    • Time Period Filters
    • From – click and select the date
    • To – click and select the date

    Figure 162 Scatter View

     

    How to Access the Scatter View

    1. Upon accessing the Insights Module (continue from How to Access the Insights Module), click on Scatter from the Correlation View area of the screen

    This section can also be accessed by clicking on a Correlation Heat Map square which indicates correlation for any 2 metrics (refer to Figure 159). It will navigate to the Scatter view for the 2 metrics that were correlated in the heat map square.
